# Git Commands Cheatsheet

## Initialize Repository
git init

## Check Status
git status

## Add Files
git add .
git add filename

## Commit Changes
git commit -m "message"

## View History
git log
git log --oneline

## View Changes
git diff
git diff --staged

## Ignore Files
Create .gitignore file
